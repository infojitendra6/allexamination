Chhattisgarh PSC Pre examination Result 2019

Name of Post: Chhattisgarh PSC Pre examination 2019
Post Date : 03 October 2018
Post Update Date : 17 April 2019
Short Information : Chhattisgarh Public Service Commission CGPSC Online Are Invited Online Application Form for the State Service Prelim Examination 2018-2019. Those Candidates Are Interested to the Following Vacancy and Completed the All Eligibility Criteria Can Read the Full Notification and Apply Online.

Short Details of Application

Application Dates

  • Application Start : 07 December 2018
  • Last Date of Application Form : 05 January 2019
  • Last Date Payment : 05 January 2019
  • Form Currection Date: 08-14 January 2019
  • Prelim Admit Card Available :07 February 2019
  • Prelim Examination Date : 17 February 2019
  • Prelim Answer Key : 01 March 2019
  • Prelim Result Declared Date : 17 April 2019
  • Main Examination Date : 21-24 June 2019

Application Fees

  • General / Other State : 400/-
  • Reserve Category : 300/-
  • Correction Charge : 15/-
  • Pay the Examination Fee Through Credit Card, Debit Card, Net Banking(Online Mode) OR Challan, Cash (Offline Mode).

Age Limit As On

  • Minimum Age : 21 Years
  • Maximum Age : 28 Years (Police Post)
  • Maximum Age : 38 Years(Other Post)
  • Age Relaxation EXtra As Per Rules(Only For Home State).

Examination Eligibility

  • Bachelor Degree in Any Stream in Any Recognized University in India
